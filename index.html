Simon is a memory game where you need to repeat a sequence of tones and lights. The device has four colored buttons, each producing a particular tone when it is pressed or activated by the device. A round in the game consists of the device lighting up one or more buttons in a random order, after which the player must reproduce that order by pressing the buttons. If the user succeeds, the series becomes progressively longer and more complex.

Game instructions

  • To start the game flag the Power Mode toggle and press the Play button.
  • By flagging the Strict toggle, you can activate the strict play mode, in this case whenever you do a mistake the game is over
  • The game has 10 turn, each turn starts with the device flashing a sequence of tones and lights.
  • The number of the flashes always matches the turn number shown in the blue counter display, next to the play button
  • As soon as the number of flashes played by the computer matches the player sequence, the computer turn is over
  • The player needs to repeat the sound by pressing the coloured button
  • If the player sequence matches the Simon's one, the following round will flash the same sequence with an additional sound
